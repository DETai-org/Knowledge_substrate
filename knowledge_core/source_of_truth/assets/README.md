# Assets

Папка `assets/` хранит технические материалы, которые используются для сборки и публикации
Source of Truth в MkDocs (JS/CSS, шаблоны и переопределения). Это не контент знаний, а
служебный код, обеспечивающий отображение и поведение сайта.

## Структура

- `javascripts/` — пользовательские скрипты MkDocs (дополнительное поведение интерфейса).
- `stylesheets/` — пользовательские стили MkDocs (визуальное оформление и правки темы).
- `overrides/` — переопределения шаблонов MkDocs Material.
  - `overrides/partials/` — частичные шаблоны (partials), подключаемые темой.
