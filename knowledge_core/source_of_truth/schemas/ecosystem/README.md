# Ecosystem Schema

`schemas/ecosystem/` хранит контракты и политики домена `ecosystem`.

Пока основная каноническая формулировка ecosystem metadata уже существует в документе:

- [Document Metadata Policy](https://detai-org.github.io/Knowledge_substrate/ru/ecosystem/Management_layer/Docs-Ecosystem/document_metadata_policy/)

Смысл этого каталога:
- не дублировать сами ecosystem-документы;
- фиксировать contracts, policies и naming rules для ecosystem-layer;
- со временем собрать здесь явный schema-layer, зеркальный `docs/{lang}/ecosystem/...`.

Текущий README выполняет роль навигационного указателя до момента, пока ecosystem schema не будет разложена на отдельные файлы.
