## Source of Truth (SoT)

**Source of Truth** экосистемы **DET / DETai** включает **канонические Markdown-документы**, их **метаданные** (frontmatter) и **Controlled vocabulary** (канонические реестры допустимых значений, например: rubrics, categories, keywords). Эти материалы являются **первичным источником знания** и используются как единый фундамент для:

- **публикации** через **MkDocs Material** (при этом публикуется только то, что мы явно включаем в навигацию/структуру через файл `mkdocs.yml`);
- **индексации** и структурирования данных в операционном хранилище (SQL/Cloud SQL);
- **построения knowledge graph** (связи, ссылки, подграфы по доменам);
- **работы интеллектуальных агентов** (поиск, RAG, embeddings и другие машинные представления).

**Controlled vocabulary** живут **внутри Source of Truth**, но **не являются частью конкретных документов**: это отдельные канонические реестры, на которые документы ссылаются через стабильные ID.  

Базы данных и сервисы (ingest pipeline, API, Postgres) являются **производными проекциями** этого канона: при необходимости они могут быть **воспроизведены заново** из Source of Truth.
