# Quotes

`docs/publications/quotes/` — место для канонических quote-документов домена `publications`.

Здесь должны жить **сами Quote Records** как публикационные документы, если они переводятся в общий canonical layer Knowledge Substrate.

Это не место для:
- user runtime data;
- brand kits;
- bot settings;
- SQL runtime state;
- производных binary artifacts как primary source.

Схема и контракт этого типа документа находятся в:
- `knowledge_core/source_of_truth/schemas/publications/quotes/quote_record_contract.md`
