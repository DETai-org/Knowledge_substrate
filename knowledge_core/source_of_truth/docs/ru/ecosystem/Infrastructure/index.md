# Infrastructure

**Инфраструктура в экосистеме DET / DETai** — это **система взаимосвязей** между **проектами**, **данными** и **инструментами**, которая создаёт **структуру**, обеспечивающую эффективную работу всех частей экосистемы. Всё, что создаётся в системе, должно быть **согласовано и интегрировано** с этой структурой, чтобы обеспечить **взаимодополняющее** взаимодействие между её компонентами.

Она необходима для того, чтобы экосистема могла **реализовывать четыре ключевых принципа**: **наглядности**, **навигации**, **системности** и **адаптивности**, обеспечивая бесперебойное взаимодействие всех её компонентов.


На данный момент, в разделе **инфраструктура** выделяются два ключевых направления:

1. **Сервера** — предоставляют вычислительные мощности и хранилища для обработки и хранения данных экосистемы.
    
2. **Knowledge_Substrate** — каноническая структура знаний и данных, которая управляет метаданными, пайплайнами и операционными хранилищами знаний экосистемы DET / DETai
