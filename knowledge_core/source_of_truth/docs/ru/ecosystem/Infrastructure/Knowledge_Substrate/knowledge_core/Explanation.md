
# Semantic Similarity Graph

Semantic Similarity Graph — это слой системы, который связывает опубликованные посты на сайте между собой на основе их смыслового сходства.

Он строится на основе канонических документов из Source of Truth и не является самостоятельным источником данных. Граф не хранит первичные смыслы, а отражает отношения между ними в виде связей по семантической близости.

Внутри системы этот слой занимает промежуточное положение между каноном и пользовательскими интерфейсами. Он использует тексты постов как входные данные, материализуется в операционных структурах (векторы и связи), доступен через API и используется внешними слоями для навигации и анализа.

На текущем этапе граф формируется только для документов типа `post` со статусом `publish`. Это ограничение задаёт минимально достаточную область применения, позволяя получить устойчивую модель без зависимости от контрактов других типов документов.

Принципиально **граф ориентирован на содержание текстов постов**, а не на их метаданные. Близость определяется смыслом, а не совпадением рубрик или категорий. Архитектура слоя остаётся расширяемой и допускает включение новых типов документов и способов представления связей в будущем. Граф существует как обновляемая операционная структура, а не как статичный артефакт.

Semantic Similarity Graph не является поисковой системой, не реализует вопросно-ответные механизмы и не заменяет таксономию. Его функция — формирование смыслового рельефа корпуса, а не извлечение ответов.

В перспективе этот слой может быть расширен за счёт подключения других типов документов, построения тематических сообществ, использования более детализированных представлений текста и автоматического формирования концептуальных узлов. Текущая реализация задаёт базовый каркас для этих направлений развития.
