
**Проект Telegram** — это часть экосистемы DET/DETai, созданный   для автоматизации взаимодействий с Telegram. В этом проекте собраны инструменты, которые позволяют управлять действиями в Telegram через пользовательские аккаунты и контентные процессы. 

### ⚙️ Архитектура и структура

**Проект Telegram** функционирует как **проект второго эшелона** в рамках экосистемы DET/DETai.
