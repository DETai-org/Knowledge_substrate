# 🧠 База знаний META

Этот файл описывает основные принципы организации работы и материалов META.

---
## 📁 Структура стандартных файлов для каждого кастомного GPT:

- `01_rules.md` — Правила поведения, логика работы агента.
- `02_knowledge_base.md` — База знаний и справочные материалы.
- `03_style_guide.md` — Гайд по стилю оформления ответов и взаимодействия.

Дополнительно могут использоваться:
- `04_integration_notes.md` — если агент связан с внешними сервисами (API, интеграции).
- `05_dev_ideas.md` — идеи для развития агента.
- `06_prompt_library.md` — коллекция примеров промтов и задач.

---

## 🔹 Принципы работы с базой знаний:

- База знаний расширяет понимание агента, но не заменяет его основную инструкцию.
- Файлы базы знаний используются при необходимости, по контексту задачи.
- Новые файлы могут добавляться в зависимости от специфики проекта.

---

## 📚 База знаний помогает:

- Стандартизировать процесс создания кастомных GPT.
- Сохранять чистоту структуры проектов.
- Ускорять развитие новых агентов.
