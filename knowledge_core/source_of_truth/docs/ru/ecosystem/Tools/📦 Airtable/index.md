# Airtable

Airtable — это инструмент для работы со структурированными данными, который можно условно представить как «умные таблицы»: что-то среднее между Excel, базой данных и простым интерфейсом для хранения информации. В отличие от обычных таблиц, AirTab позволяет задавать типы полей (статус, версия, файл, ссылка), работать с представлениями (таблица, канбан, списки) и использовать данные как единый структурированный набор, а не просто как текст.

В экосистеме DETai AirTab используется как **возможно временный рабочий слой контекста** для взаимодействия с ChatGPT. Он позволяет хранить небольшой набор актуальных документов (стандарты, политики, рабочие файлы), чтобы не передавать их вручную в каждом чате. Airtable не является базой знаний и не заменяет Knowledge Storage (SQL / GitHub), а служит вспомогательным инструментом на переходном этапе — до появления полноценного агентского слоя доступа к знаниям.
