

# Airtable в экосистеме DETai 

## Роль инструмента

Airtable в экосистеме DETai используется сейчас как **временный инструмент доступа к контексту** для ChatGPT на этапе, когда полноценный агентский слой ещё не реализован.

Он не является:
- базой знаний,
- Source of Truth,
- заменой Knowledge Storage,
- долговременным хранилищем.

## Назначение использования

Airtable применяется для:
- хранения небольшого количества актуальных файлов,
- стандартов и политик, необходимых «здесь и сейчас»,
- уменьшения количества ручных загрузок файлов в чаты,
- ускорения работы в переходный период.

Пока агент не готов, мы можем использовать Airtable как временный внешний контекст для ChatGPT, чтобы не таскать файлы руками.
Это не замена нашей базе знаний, а вспомогательный рабочий слой.
