**FileZilla** — это **инструмент передачи файлов** (FTP/SFTP).

## 📦 Что делает FileZilla

**FileZilla** — это:

- клиент для FTP / SFTP
- «перетащить файл туда-обратно»
- быстрые ручные операции с сервером

**Хорош для:**

- загрузить файл
- скачать лог
- заменить конфиг
- разовая правка

### В нашей экосистеме:

👉 **VS Code + SSH вытесняет FileZilla почти полностью.**

FileZilla остаётся:

- как аварийный инструмент
- как «ручной кран»
- как временный мост
