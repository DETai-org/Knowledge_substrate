# Technical Standards

Здесь хранятся документы, которые описывают **форматы**, **правила** и **нормы** работы экосистемы DET, а также определяют **когда и как должны вноситься изменения** в эти нормы.

Основные типы документов для этой папки: **Standard** и **Policy** 

 Важно помнить, что **тип не всегда равен папке**, и документы могут быть добавлены в зависимости от потребностей экосистемы.
