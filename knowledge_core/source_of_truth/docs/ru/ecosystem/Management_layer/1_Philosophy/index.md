
# Philosophy

Cдесь хранятся документы, которые имеют отношение к философским и концептуальным аспектам системы, её основным установкам и принципам.

Эта папка **в основном** будет содержать документы типов:

- **Philosophy** — документы, которые фиксируют основополагающие идеи, ценности и мировоззрение экосистемы.
    
- **Principles** — документы, которые конкретизируют основные установки и правила, регулирующие деятельность в экосистеме.


Также могут быть добавлены другие документы.
