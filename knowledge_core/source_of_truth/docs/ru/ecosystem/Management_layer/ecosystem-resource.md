# 💰 Ресурсы экосистемы

Ресурсная модель описывает, за счёт каких базовых ресурсов живёт, растёт и масштабируется экосистема DET/DETai. Каждый проект, независимо от эшелона, должен усиливать хотя бы один из этих ресурсов — финансовый, командный, технологический, репутационный или аналитический.

Эта модель позволяет оценивать вклад проектов не абстрактно, а через конкретный прирост системы: деньги, людей, технологии, узнаваемость или данные. Таким образом, ресурсы становятся измеримым фундаментом устойчивого развития платформы.

# Ресурсная модель (5 базовых ресурсов)

> Круг = ресурс. Для каждого — краткое определение, «векторы внутри», фишка и как меряем.



Принцип ресурсного вклада
📘 Проект считается завершённым, если он дал прирост хотя бы в одном из этих направлений.

| Ресурс                          | Что означает                          |
| ------------------------------- | ------------------------------------- |
| 💰 **Финансовый**               | Генерация дохода или экономия         |
| 👥 **Командный**                | Рост вовлечённости, сотрудничества    |
| ⚙️ **Технологический**          | Создание новых инструментов и решений |
| 🟡 **Ресурс узнаваемости (PR)** | Расширение аудитории и бренда         |
| 📊 **Аналитический**            | Улучшение систем понимания и метрик   |


## 💰 Финансы

**Что это:** приток и управление деньгами: продажи, подписки, донаты, гранты, партнёрства.  
**Векторы внутри:** продажи сервисов/ботов, консалтинг, подписки/донаты, гранты, B2B-лицензии.  
**Фишка:** позволяет нанимать людей, расширять инфраструктуру и маркетинг.  
**Как меряем:** MRR/ARR, средний чек, LTV, CAC, ROI, доля реинвестиций, runway (мес.).

## 👥 Команда (человеческий ресурс)

**Что это:** люди, роли и компетенции 



**Векторы внутри:** ядро, подрядчики, волонтёры/студенты, соавторы.  
**Фишка:** мультипликатор — ускоряет создание проектов и рост остальных ресурсов.  В идеале чтобы за каждый ресурс отвечал 1 человек который формирует вокруг себя небольшую команду. Это создаёт аналог «отделов» внутри ИМПЕРИИ 😄
**Как меряем:** FTE/ролей закрыто, скорость спринтов, bus-factor, % внешней помощи.

См. [[bee-roles]]

## ⚙️ Технологии

**Что это:** инфраструктура и инновации, которые конвертируются в эффективность.  
**Векторы внутри:**

- **Операционные мощности** (серверы, дроп-машины);
- **AI-интеграции** (🧭Codex, Ⓜ️Make Сценарии, Copylot, );

**Фишка:** делает «больше за меньшее» — автоматизирует, ускоряет, стабилизирует.  

**Как меряем:** аптайм, MTTR, время деплоя, % автоматизации, стоимость инфры/польз., скорость вывода фич.

## 🟡 PR / Узнаваемость

**Что это:** видимость бренда и персоналий (охват, репутация, доверие).  
**Векторы внутри:** контент-план, рассылки/блог, выступления, видео Youtube.  
**Фишка:** привлекает внимание → людей → деньги → исследования.  
**Как меряем:** подписчики, ER, охват/показы, трафик на сайт/ботов, конверсия в активацию подписок.

## 📊 Аналитика / данные

**Что это:** обезличенные данные из проектов/ботов + ресёрч; «топливо» для статей и улучшений.  
**Векторы внутри:** event tracking, исследовательские датасеты, конверсии, когортный анализ.  
**Фишка:** замыкает цикл: **данные → публикации → научная ценность → усиление метода**.  
**Как меряем:** валидные сессии/дни, покрытие событий, число датасетов/пользователей конкретного бота/проекта

# Базовые связи (по умолчанию)

- **Проекты (Э1, Э2)** → **📊 Аналитика/данные** → статьи → усиливают **Концепцию/Метод**.
    
- **💰 Финансы** → растят **👥 Команду** и **⚙️ Технологии**.
    
- **🟡 PR** → растит трафик и доверие → конвертируется в 💰 и 👥.
    
- **⚙️ Технологии** → ускоряют всё: Э1/Э2 выпускаются быстрее и стабильнее.

# Шкала для весов (0–3)

- **0** — нет влияния
    
- **1** — низкое
    
- **2** — среднее
    
- **3** — высокое
