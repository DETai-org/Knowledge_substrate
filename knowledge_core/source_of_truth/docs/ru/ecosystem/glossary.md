
# Глоссарий

**Epic Issue** — это GitHub Issue, представляющая собой **документ осмысленного перехода** проекта или репозитория в новое состояние.  
Epic Issue фиксирует смысл, рамку и критерии этого перехода и служит контейнером для связанных Sub-Issue, совокупное выполнение которых приводит систему к новому функциональному состоянию.
