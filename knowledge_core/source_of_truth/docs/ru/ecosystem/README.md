# Ecosystem

Канон знаний по экосистеме DET / DETai.

## Навигация MkDocs для редакторов

Этот раздел (`docs/ru/ecosystem/`) публикуется в MkDocs и использует **автоматическую навигацию** на базе структуры директорий.
Порядок страниц и разделов в левом сайдбаре настраивается через файлы `.pages`, которые лежат рядом с материалами разделов.

### Как менять порядок страниц

Если нужно изменить порядок отображения:

1. Найдите нужный раздел в дереве `docs/ru/ecosystem/`.
2. Откройте соответствующий файл `.pages` в этом разделе.
3. Отредактируйте блок `nav:` (порядок страниц и подпапок).
4. При необходимости используйте `hide: true` для скрытия целого раздела или уберите файл из списка `nav:`.

> Для корневого раздела `docs/ru/ecosystem/` служебный файл `README.md` исключён из сайдбара и не добавляется в `nav:`.
