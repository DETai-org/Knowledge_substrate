# Ecosystem

Канон знаний по экосистеме DET / DETai.

## Источник структуры и последовательности

Полная, актуальная и углублённая спецификация порядка папок и файлов вынесена в отдельный файл:

- [`sortspec.md`](./sortspec.md)

Именно его нужно использовать как общий источник истины для:

- синхронизации структуры `ru/ecosystem`;
- поддержки порядка в Obsidian (sorting-spec);
- синхронизации навигации MkDocs (`mkdocs.yml`).

> `sortspec.md` намеренно **не публикуется в MkDocs** и используется только как внутренний технический артефакт.
