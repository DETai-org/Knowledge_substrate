# Git для дураков но умнеющих


**Вступление:**

> Да, я не Джуниус Максимус по Git, и `rebase` звучит как латинский глагол. Но я учусь. Этот файл — наш персональный шпаргалон, где  не стесняюсь объяснять себе простыми словами, что за фигня тут происходит и почему оно работает (или нет). Всё, чтобы не гуглить одно и то же 78-й раз.

| Команда       | Что добавляет                                               | Не добавляет                            | Когда использовать                                                |
|---------------|-------------------------------------------------------------|-----------------------------------------|-------------------------------------------------------------------|
| `git add -u`  | Изменённые и удалённые файлы, которые уже отслеживаются     | Новые (неотслеживаемые) файлы           | Если хочешь обновить только те файлы, за которыми Git уже следит |
| `git add .`   | Новые и изменённые файлы в текущей директории и подпапках   | Удалённые файлы                         | Если добавил новый файл, но не удалял ничего                      |
| `git add -A`  | Новые, изменённые и удалённые файлы — всё подряд            | Игнорируемые файлы (.gitignore)        | Если хочешь зафиксировать все изменения целиком                  |


важный момент: когда не видишь обновлений на сервере попробуй создать новую вкладку иногда кеш **браузера не отображает новые файлы**
