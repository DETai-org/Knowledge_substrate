#  Ⓜ️Make


Make — это платформа для визуальной автоматизации рабочих процессов, которая позволяет соединять различные приложения и сервисы без написания кода.

**Как это работает:**

Вы строите сценарии (scenarios) в визуальном редакторе, соединяя модули — готовые блоки, которые выполняют конкретные действия: получить данные из одного сервиса, обработать их, передать в другой сервис.

**Чем отличается от конкурентов:**

В отличие от Zapier, где логика линейная и ограниченная, Make даёт гораздо больше гибкости. Вы можете:
- Создавать ветвления (if/then)
- Организовывать циклы (iterators)
- Работать с агрегаторами данных
- Обрабатывать массивы
- Настраивать маршрутизацию данных между модулями

**Какое место занимает в стеке инструментов:**

По сути Make заменяет написание кода для интеграций. То, что раньше требовало разработки отдельного скрипта или микросервиса, теперь собирается визуально за часы, а не дни. По концепции это аналог n8n, но в облачной версии и с более проработанным визуальным интерфейсом.
