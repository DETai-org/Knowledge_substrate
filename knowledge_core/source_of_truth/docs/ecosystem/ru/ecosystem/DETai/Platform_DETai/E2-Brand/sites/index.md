# Sites

**sites** — это monorepo веб-проектов DET/DETai. Он содержит публичные Next.js-сайты экосистемы, включая `detai-site` и `personal-site`, и служит витриной для проектов, публикаций, юридических страниц и пользовательских web-сценариев.

## Структура и архитектура

Проект построен как **Next.js monorepo** и размещается через **Vercel**. Такая структура позволяет держать несколько сайтов в одном репозитории, но разворачивать их как отдельные Vercel Projects.

- **Next.js** — это **фреймворк** для создания сайтов и веб-приложений, который поддерживает **серверный рендеринг**, **оптимизацию изображений**, **SEO** и другие возможности, что делает сайт быстрым и удобным для пользователей.

- **Vercel** — это **хостинг для фронтенда**, который обеспечивает **автоматический деплой** через GitHub и высокоскоростную доставку контента по всему миру с помощью **CDN**.

Проект использует эти технологии для обеспечения **мгновенной доступности сайтов** и лёгкости в управлении контентом, не требуя ручного обновления production-среды.

## Значение для экосистемы DET/DETai

`sites` является неотъемлемой частью **глобальной стратегии** экосистемы DET/DETai. Это **не просто блог**, а **web-слой**, который интегрирует **многоязычие**, **системы AI**, публикационные материалы, проектные страницы и юридические документы, создавая понятное публичное присутствие для широкой аудитории.

## Как работает сайт

- **Динамическая генерация**: Сайт использует **серверный рендеринг** (SSR), что позволяет страницам загружаться быстро и быть доступными для поисковых систем.

- **Многоязычие**: Проект поддерживает несколько языков, обеспечивая локализацию контента для разных регионов.

## Почему это важно

`sites` — это **не просто инструмент для отображения контента**, а ключевая **платформа для взаимодействия с аудиторией**, которая поддерживает и развивает публичные процессы экосистемы. Это **интерактивная среда**, где каждый пост, проектная страница и версия контента становятся частью **глобальной сети**, доступной пользователям в разных языковых и продуктовых контекстах.

`sites` становится важнейшим **инструментом для маркетинга**, **образования** и **публикаций**, который помогает строить присутствие и развивать экосистему DET/DETai.
