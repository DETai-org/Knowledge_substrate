# Concept Fact

 Этот раздел предоставляет **введение** в теоретическую основу и описывает ключевые **постулаты** DET, которые сейчас находятся на стадии развития, и представляют собой фундамент для будущего метода.

---

## Данности:

1. DET на данный момент не является методом в классическом понимании, так как у него отсутствуют конкретные методологические инструменты. Это концепция, которая находится в процессе формирования

2. Концепции рождается из диссертации и размышлений

3. Пока нет конкретных техник, это не метод, а концептуальная основа для будущего психотерапевтического инструмента.

4. **Продукт**: бедующая книга — это **шаг** в создании теоретической базы.

5. Метод может строится только на концептуальной основе. На данный момент метод ещё находится в стадии очертаний, но база для его разработки уже подготовлена.

6. **DET** в будущем станет в ряд с такими методами, как **DBT**, **TFP** и другими признанными психотерапевтическими подходами.
