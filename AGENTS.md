# Инструкции для агента

- Если изменения затрагивают MkDocs (например, `mkdocs.yml`, структуру страниц, навигацию или плагины), сначала убедись, что зависимости установлены из `requirements.txt`, и запускай `mkdocs build --clean` для проверки сборки.
- Держи `requirements.txt` в актуальном состоянии: добавляй туда зависимости, которые нужны для сборки MkDocs.
